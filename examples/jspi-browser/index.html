About This Demo

This runs the Glulxe interpreter compiled to WebAssembly (WASI), using JSPI (JavaScript Promise Integration) to handle async I/O in the browser.

Try commands like: look, go north, inventory, help

Requirements

JSPI requires Chrome 131+ (enabled by default) or earlier versions with the flag: chrome://flags/#enable-experimental-webassembly-jspi
